# Simple Calculator Project

## Project Title
Simple Calculator - A console-based calculator supporting addition, subtraction, multiplication, and division.

## Description
A beginner-level Python project that demonstrates basic I/O, functions, error handling, and use of Git & GitHub. Designed for first-year students.

## Features
- Add, subtract, multiply, and divide two numbers.
- Input validation for numbers.
- Division-by-zero handling.
- Simple menu-driven console UI.
